# Handoff: Ebu.Gubkin — редизайн (glassmorphism)

## Overview
Ebu.Gubkin — неофициальная студенческая платформа РГУ нефти и газа им. Губкина.
Разделы: **Форум**, **Биржа** (фриланс между студентами), **ГОСТ-калькулятор**,
**Кошелёк**, **Профиль**. Этот пакет — утверждённый редизайн в стиле
**glassmorphism** (глубокий фиолетовый градиент + полупрозрачные «стеклянные»
карточки), целевая аудитория — студенты 18–25 лет.

## About the Design Files
Файлы в этом пакете — **дизайн-референсы, сделанные в HTML/CSS**. Это прототипы,
показывающие желаемый вид и поведение, **а не продакшн-код для копирования**.
Задача — **воссоздать эти экраны в существующем окружении вашей кодовой базы**
(React/Vue/Next и т.п.), используя ваши паттерны, компоненты и библиотеки. Если
окружения ещё нет — выбрать подходящий фреймворк (рекомендуется React + CSS-in-JS
или CSS Modules / Tailwind) и реализовать дизайн там.

Прототипы построены как набор JS-модулей, каждый из которых возвращает
`{ css, html }` строку, и собираются на «дизайн-канвасе» (`design-canvas.jsx`)
только для удобного просмотра. **Канвас в продакшн не нужен** — берите из модулей
вёрстку и стили.

## Fidelity
**High-fidelity (hifi).** Финальные цвета, типографика, отступы, радиусы, тени и
состояния указаны точно. Воссоздавайте UI пиксель-в-пиксель средствами вашего
кодовой базы. Контент (имена, тексты тем, суммы) — демонстрационный, заменяется
реальными данными.

### Glass-поверхность (базовая карточка)
```css
.glass{
  background: rgba(255,255,255,.06);
```

`transform: translateY(-3px)` или `translateX(3px)`, transition `.16s`.

## Валютная модель (ВАЖНО)
- **₽ (рубли)** — единственная валюта на форуме, бирже, в профиле и кошельке:
  баланс, бюджеты заказов, заработок, награды за ответы, переводы, история
  транзакций.
- **₮ (токены)** — **только** в ГОСТ-калькуляторе. Токены покупаются с рублёвого
  баланса кошелька по курсу **1 ₮ = 10 ₽**. В истории кошелька это видно одной
  строкой «Покупка ГОСТ-токенов» (списание в ₽).
- Не использовать токены как валюту вне ГОСТ-калькулятора.

## Screens / Views

### 1. Тема на форуме — `directions/screen-thread.js`
**Purpose:** чтение треда, реакции, ответ.
**Layout:** хлебные крошки (Форум / Категория / Тема), затем grid `1fr 300px` (контент + сайдбар), gap 20px, padding 0 26px 40px.
**Компоненты:**
- **Шапка темы** (`.th-head`, glass, radius 24, padding 26×28): теги-пилюли (#термех — заливка lav-градиент; вторичные — `rgba(255,255,255,.1)`, текст lav); H1 30px/700; byline = аватар 44px + имя/мета + блок статов справа (Ответы / Просмотры (mint) / Реакции (gold)).
- **Пост** (`.post`, glass, radius 20, flex gap 16). Пост автора (`.op`) выделен рамкой `rgba(196,181,253,.35)` и фоном `rgba(124,58,237,.12)`. Внутри: аватар 46px, шапка (имя + бейдж роли + время справа), текст (`code` — мятный на `rgba(255,255,255,.1)`), опц. цитата (левый бордер 3px lav), строка реакций.
- **Бейджи ролей:** «Автор» — золотой фон `--gold`; «Ментор» — мятный `--mint`; текст `#08221c`.
- **Реакции** (`.rx`): пилюля `rgba(255,255,255,.07)` + border; активная (`.act`) — `rgba(94,234,212,.16)` + border mint, текст mint. Справа текстовые ссылки: Ответить / Пожаловаться / Поделиться (`.rxlink`, `--dim`, hover→`--ink`).

- **Сайдбар:**

Пакеты (`.pack`): 10 ₮ — 100 ₽; **60 ₮ +20%** (50 + 10 бонус, `.best` мятная рамка) — 500 ₽; 130 ₮ +30% (100 + 30) — 1 000 ₽. `pk-t` — кол-во токенов ₮ + бонус-тег; `pk-p` — цена в ₽ mint. ## Interactions & Behavior
- **Навигация:** клик по пунктам шапки → соответствующий раздел. Аватар → профиль. Пилюля кошелька → кошелёк.
- **Форум:** клик по теме → страница темы; реакции переключаются (toggle, состояние `.act`); «Создать тему»/«Отправить ответ» → отправка формы.
- **Биржа:** сегмент «Заказы/Услуги» переключает ленту; чипы категорий фильтруют; клик по карточке → детальная страница заказа/услуги; «Разместить заказ» → форма.
- **Профиль:** табы переключают контент; «Подписаться»/«Написать» → действия.
- **Кошелёк:** «Пополнить/Вывести/Перевести» → модалки; пакеты токенов выбираются (выделение `.best`), «Купить с баланса» → списывает ₽ по курсу 1 ₮ = 10 ₽; фильтры истории.
- **Hover:** карточки приподнимаются (`translateY(-3px)` / `translateX(3px)`), осветляются (`rgba(255,255,255,.1)`), transition `.14–.18s`.
- **Адаптив:** макеты заданы под ~1320px ширины. Для мобильных колонки `1fr Npx` сворачиваются в одну колонку, шапка → бургер-меню, сетки карточек → 1 колонка.
