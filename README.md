# Ebu.Gubkin — Студенческий портал Губкинского университета

Ebu.Gubkin — единый веб-портал для студентов РГУ нефти и газа имени И.М. Губкина.  
Объединяет четыре независимых сервиса под одним аккаунтом и общим кошельком.

---

## Разделы

### Биржа
Студенческая биржа учебных услуг. Студенты размещают заказы (написание рефератов, решение задач, помощь с курсовыми), исполнители откликаются и берут их в работу. Встроенный чат между заказчиком и исполнителем. При спорных ситуациях подключается администратор и разрешает спор вручную.

Основные сущности: заказ, объявление исполнителя, отклик, чат, спор, отзыв.

### Форум
Тематический форум, разбитый по категориям (предметы, жизнь универа, объявления и др.). Студенты создают темы и отвечают на посты. Встроена двухуровневая модерация: быстрая фильтрация по регулярным выражениям и отложенный AI-анализ DeepSeek для выявления спама и нарушений.

### ГОСТ-калькулятор
Сервис для автоматической генерации технических документов по ГОСТ на основе загруженных исходников (PDF, Word). Три режима работы: универсальный, по фиксированному шаблону, по пользовательскому шаблону. Оплата токенами, которые покупаются за рублёвый баланс.

### Кошелёк
Единый финансовый центр. Хранит рублёвый баланс (для Биржи) и ГОСТ-токены на одном экране. Пополнение — через ручное подтверждение администратором. Вывод — заявка с указанием карты. Реферальная программа: бонус 5% от суммы первых трёх пополнений приглашённого пользователя.

---

## Единый аккаунт

Пользователь регистрируется один раз. Один профиль даёт доступ ко всем разделам. Профиль содержит никнейм, аватар, контактные данные, рейтинги (как заказчика и как исполнителя), реферальный код и историю всех финансовых операций.

## Стек

**Frontend**
- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router v6
- Supabase JS SDK
- Lucide React (иконки)

**Backend — Биржа**
- Node.js + Express.js
- Supabase (сервисный клиент)
- Multer (загрузка файлов)
- DeepSeek API (AI-модерация форума)
- Telegram Bot API (уведомления для администратора)
- Задеплоен на Render

**Backend — ГОСТ-калькулятор**
- Python + FastAPI
- Отдельный сервис, задеплоен на Render

**База данных и инфраструктура**
- Supabase: PostgreSQL, Auth, Storage, Realtime
- GitHub Pages (хостинг фронтенда)
- GitHub Actions (CI/CD)

## Роли пользователей

| Роль | Возможности |
|---|---|
| **Гость** | Просмотр ленты заказов, объявлений, форума |
| **Пользователь** | Всё выше + создание заказов, отклики, чат, кошелёк, ГОСТ |
| **Администратор** | Всё выше + admin-панель: подтверждение платежей, разрешение споров, бан пользователей, модерация форума, настройки сайта |

## Деплой

Push в ветку `main` или `master` автоматически запускает сборку и деплой на GitHub Pages через GitHub Actions.
